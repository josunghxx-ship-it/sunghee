# SUNG

조성희 포트폴리오 웹사이트 소스입니다.

## 파일 구성

- `index.html`: 실제 배포에 사용하는 HTML
- `assets/css/styles.css`: 화면 스타일
- `assets/js/main.js`: 메뉴, 스크롤, 등장 효과
- `portfolio.html`: Claude로 만든 원본 초안
- `projects/join-our-team/`: 해커톤 팀원 모집 모바일·태블릿 페이지

`portfolio.html`은 비교와 복구를 위한 원본으로 유지하고, 이후 수정은 분리된 배포용 파일을 기준으로 진행합니다.
